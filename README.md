# Knowledge Base

本地知识库桌面应用 - 全文搜索、双向链接、知识图谱

基于 Tauri 2.x 构建的桌面应用。

## 开发

```bash
pnpm install
pnpm tauri dev
```

## 构建

```bash
pnpm tauri build
```
